MessageSummary Class

A summary of a message.

Syntax

public class MessageSummary : IThreadable

Remarks

A MailKit.MessageSummary is returned by IFolder.Fetch(UniqueId[], MessageSummaryItems, System.Threading.CancellationToken). The properties of the MailKit.MessageSummary that will be available depend on the MailKit.MessageSummaryItems passed to the aformentioned method.

Syntax

Value

The body structure of the message.

Remarks

The body will be one of MailKit.BodyPartText, MailKit.BodyPartMessage, MailKit.BodyPartBasic, or MailKit.BodyPartMultipart.

Envelope Property

Gets the envelope of the message, if available.

Syntax

public Envelope Envelope { get; }

Value

The envelope of the message.

Remarks

The envelope of a message contains information such as the date the message was sent, the subject of the message, the sender of the message, who the message was sent to, which message(s) the message may be in reply to, and the message id.
